# 넷플릭스 스타일 메인 페이지 클론

## 📌 프로젝트 개요

넷플릭스의 메인 UI를 HTML/CSS로 재현한 프로젝트입니다. 시맨틱 태그를 활용한 구조적 마크업과 반응형 디자인을 구현했습니다.

## 🎯 구현 기능

### 1. 고정형 헤더 (Sticky Header)

- 스크롤 시에도 상단에 고정되는 내비게이션
- 로고, 메뉴, 검색/알림/프로필 버튼 포함
- `position: sticky`를 활용한 자연스러운 고정 효과

### 2. 다중 그라데이션 메인 배너

- 좌우/상하 이중 그라데이션으로 텍스트 가독성 확보
- 패럴랙스 효과 (`background-attachment: fixed`)
- 재생/상세정보 버튼 포함
- Unsplash 고품질 배경 이미지 적용

### 3. 6열 콘텐츠 카드 섹션

- Flexbox 기반 가로 스크롤 구현
- 2:3 비율의 넷플릭스 포스터 형태 (`aspect-ratio: 2/3`)
- 호버 시 확대 효과 (`transform: scale(1.05)`)
- 스크롤바 완전 숨김 처리
- 총 30개의 콘텐츠 카드 (섹션당 10개)

### 4. 푸터

- 3단 컬럼 레이아웃 (Grid 사용)
- 소셜 링크 및 회사 정보
- 반응형 디자인 (모바일에서 1단 컬럼)

## 🔧 제출 전 보완 사항

### 1. 배너 가시성 개선

**문제**: AI가 생성한 배너의 그라데이션이 너무 진해 배경 이미지가 가려짐

**해결**:

```css
/* 수정 전 */
linear-gradient(to right, rgba(0, 0, 0, 0.7) 0%, rgba(0, 0, 0, 0) 60%)

/* 수정 후 */
linear-gradient(to right, rgba(0, 0, 0, 0.9) 0%, rgba(0, 0, 0, 0.3) 50%, rgba(0, 0, 0, 0.9) 100%)
```

- 중앙 부분의 투명도를 높여 배경 이미지가 보이도록 조정
- 양쪽 끝은 어둡게 유지하여 텍스트 가독성 확보

### 2. UX 최적화 (카드 호버 효과)

**문제**: 카드 확대 배율이 과도하여 인접 콘텐츠를 가림

**해결**:

- `transform: scale(1.2)` → `scale(1.05)`로 조정
- `z-index: 10` 추가로 레이어 관리
- 부드러운 전환 효과 (`transition: 0.3s ease`)

### 3. 시각적 암시 추가

**문제**: 가로 스크롤 가능 여부가 명확하지 않음

**해결**: CSS 가상 요소로 양 끝 그라데이션 추가

```css
.content-section::before,
.content-section::after {
  content: "";
  position: absolute;
  background: linear-gradient(
    to left,
    rgba(20, 20, 20, 0),
    rgba(20, 20, 20, 0.8)
  );
}
```

- 슬라이더 양 끝에 페이드 효과로 다음 콘텐츠 암시

## 💬 AI 활용 프롬프트 로그

### 1단계: 초기 구조 생성

```
"넷플릭스 스타일의 메인 페이지를 만들어줘.
헤더, 히어로 배너, 콘텐츠 카드 섹션, 푸터를 포함해줘."
```

**결과**: 기본 HTML/CSS 구조 생성 (시맨틱 태그 활용)

### 2단계: 이미지 및 비율 수정

```
"현재 content-card 섹션의 이미지가 보이지 않아.
'오징어 게임', '더 글로리' 등 각 콘텐츠에 어울리는 임시 이미지 URL(Unsplash)을 넣어주고,
이미지 비율을 넷플릭스 포스터처럼 세로가 긴 비율로 고정하는 CSS를 작성해줘."
```

**결과**: Unsplash 이미지 URL 적용, `aspect-ratio: 2/3` 설정

### 3단계: 가로 슬라이딩 구현

```
"현재 콘텐츠 카드 섹션의 UI를 넷플릭스와 더 유사하게 개선해줘.
콘텐츠 카드의 크기가 너무 커. 한 줄에 카드가 6개씩 보이도록 크기를 조정하고,
좌우 슬라이딩(가로 스크롤)이 가능하도록 CSS overflow-x: auto를 적용해줘.
슬라이딩 효과를 제대로 확인할 수 있도록 각 섹션당 10개로 늘려줘."
```

**결과**: Flexbox 레이아웃 + 가로 스크롤, 카드 30개로 확장

### 4단계: 배너 가독성 개선

```
"메인 배너(hero-banner)의 배경에 어두운 이미지나 그래디언트를 추가해서
흰색 글씨가 더 잘 보이게 수정해줘.
넷플릭스처럼 배경 이미지 위에 검은색 오버레이가 깔린 느낌이면 좋겠어."
```

**결과**: 다층 그라데이션 적용, 패럴랙스 효과 추가

### 5단계: 배너 투명도 미세 조정

```
"배너의 그라데이션이 너무 진해서 배경 이미지가 안 보여.
상단은 투명하게 하고 아래쪽만 은은하게 어두워지도록 수정해줘."
```

**결과**: 그라데이션 투명도 수치 조정 (0.7 → 0.3)

### 6단계: 카드 호버 효과 최적화

```
"호버 시 카드가 너무 크게 확대되어 옆 카드와 겹쳐.
적당한 크기인 1.05배로 줄이고 부드럽게 보이게 해줘."
```

**결과**: `scale(1.2)` → `scale(1.05)`, z-index 관리

### 7단계: 미니멀 디자인 적용

```
"현재 카드 하단의 회색 박스와 글자가 넷플릭스 느낌과 너무 달라.
배경을 제거하고, 설명(description)은 완전히 없애서 이미지에만 집중할 수 있게 해줘.
카드 사이의 간격을 줄이고, 카드 자체의 패딩을 최소화해서 이미지가 더 크게 돋보이게 수정해줘."
```

**결과**: 카드 배경 제거, description 삭제, 간격 축소 (10px → 5px)
